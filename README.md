# Pas2Go

## Документация
- `docs/overview.md` — архитектура и обзор
- `docs/usage.md` — запуск и использование
- `docs/language.md` — поддерживаемый поднабор PascalABC
- `docs/testing.md` — тестирование

## Перед началом работы
1) Подтяни последние изменения:
```
git pull --rebase
```

## Запуск
1) Установи зависимости:
```
pip install -r requirements.txt
```
2) Запусти веб-приложение:
```
bash run.sh
```
Открой в браузере адрес, который выведет Flask (обычно `http://127.0.0.1:5000`).

## После изменений
Не забудь сохранить изменения в репозитории:
```
git status -sb
git add -A
git commit -m "Описание изменений"
git push
```
