# Тестирование

## Запуск автотестов
```
python3 -m unittest discover -s tests -v
```

## Что проверяется
Позитивные сценарии:
- логические операторы
- функции и процедуры
- ветвления и циклы
- базовые типы
- массивы

Негативные сценарии:
- необъявленные переменные
- несовместимые типы
- неверные аргументы функций/процедур
- неправильная индексация массивов
- недопустимые границы массивов
