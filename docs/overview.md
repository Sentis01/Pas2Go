# Обзор проекта

## Что делает Pas2Go
Pas2Go — учебный транслятор из подмножества PascalABC в Go. Проект состоит из:
- лексера,
- синтаксического анализатора (парсер),
- семантического анализатора,
- генератора кода Go,
- веб-интерфейса на Flask.

Цель: показать полный путь от исходного текста до итогового Go‑кода с минимальным набором языковых конструкций.

## Архитектура
Пайплайн обработки:
1) Лексер → список токенов.
2) Парсер → AST.
3) Семантический анализ → проверка типов, объявлений, сигнатур.
4) Генератор → Go‑код.

Веб‑слой (Flask) принимает исходный код, прогоняет пайплайн и показывает:
- Go‑код,
- дерево разбора,
- список токенов.

## Структура репозитория
- `code/translator/` — ядро транслятора (lexer, parser, semantics, codegen).
- `code/webapp/` — веб‑интерфейс (Flask + шаблоны).
- `tests/` — автотесты (unittest).

## Поддерживаемый поднабор PascalABC (на текущий момент)
- Логические операторы: `and`, `or`, `not`, `xor`
- Типы: `integer`, `float`, `string`, `char`, `bool`
- Присваивания
- Условные конструкции: `if ... then ... else`
- Циклы: `while`, `for ... to/downto`, `repeat ... until`
- `case ... of`
- Процедуры и функции (MVP)
- Массивы 1D: `array[low..high] of <type>`

Ограничения и детали — в `docs/language.md`.
